# Do Not Invent

## Project

Hanover Accountants & Advisors Website Rebuild

## Purpose of This Document

This document defines the factual guardrails for the Hanover Accountants & Advisors website rebuild.

The goal is to create a strong, search-friendly, conversion-focused website without adding unsupported claims, fake credentials, fake locations, fake testimonials, or unverifiable statements.

This document must be followed by Claude Project, Claude Code, developers, writers, editors, and anyone creating content for the project.

---

# 1. Core Rule

Do not invent anything that has not been verified from the current Hanover website, provided by the client, or confirmed through an approved source.

If a detail is not verified, do one of the following:

```txt
Leave it out.
Use general language.
Mark it as needing client confirmation.
Ask for confirmation before publishing.
```

Do not publish assumptions as facts.
