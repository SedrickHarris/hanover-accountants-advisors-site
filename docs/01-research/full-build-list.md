# Full Build List

## Project

Hanover Accountants & Advisors Website Rebuild

## Purpose of This Document

This document defines the full website build list for Hanover Accountants & Advisors.

The build includes:

* Core pages
* Hub pages
* Service pages
* Audience pages
* Audience + service pages
* Service area pages
* Nationwide + service pages
* Local + service pages
* Industry pages
* Resource pages
* FAQ pages
* Conversion pages
* Technical SEO pages and assets

The site will position Hanover as a nationwide online accounting, bookkeeping, tax preparation, payroll, outsourced CFO, and advisory firm with strong local service-area visibility in Georgia and Virginia markets.

---

# 20. Final Build Rule

This full build list is approved for planning, scaffolding, and Claude Project setup.

Every page should be built with unique, useful, human-first content.

Do not create thin pages.

Do not create fake local office claims.

Do not duplicate content with only city or audience swaps.

Do not invent credentials, reviews, pricing, guarantees, client results, or office details.

The full website should be built as a scalable, data-driven static Next.js site ready for Cloudflare Pages deployment.
